Examples on the site

All examples are provided by the JSON Schema test suite. You are welcome to contribute your own schemas and data.

Software used

This application uses the json-schema-validator Java library. While it is here used in the context of a Web application, it has no dependencies on any Web framework. You are free to use it with any framework you want (Spring, Google Guice, Play...), or even with another protocol than HTTP entirely.

The two main dependencies of this library are Jackson and Guava. For additional validations, it also uses Joda Time and Google's libphonenumber.

This application's source code is available here. It uses Jetty as a servlet container and jQuery as a JavaScript library.
