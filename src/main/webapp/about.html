Software used

This web site uses json-schema-validator Java library (currently, version 1.99.8) as a validator engine. This library fully supports draft v3 and draft v4 validation.

This web site's source code is available here. It uses Jetty as a servlet container and jQuery as a JavaScript library.

Reporting problems

If you spot a problem on this site, either open an issue on this application's GitHub repository, or if you do not have an account on GitHub, you can email me directly.
